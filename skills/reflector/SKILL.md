# Reflector

Look back over the conversation in this session and reflect honestly. The point is to learn, not to self-congratulate — so focus on what went wrong and how to keep it from recurring.

## What to surface

Scan the session log for:

- **Friction** — moments where the user was inconvenienced, frustrated, or had to repeat themselves.
- **Directed corrections** — anything the user explicitly asked you to reflect on or do differently.
- **Defects** — output that was wrong, incomplete, or broken.
- **Misjudgments** — decisions that, in hindsight, took the wrong direction.

## What to record

For each item, write a memory entry containing:

1. **What happened** — the specific lapse, concretely.
2. **Why it happened** — the underlying cause, not just the symptom.
3. **Recurrence-prevention measure** — the concrete rule or check that would stop it next time.

Persist these to the memory layer so future sessions can apply the lesson rather than relearn it. Pair every recorded mistake with its prevention measure — a lesson without a fix is just a complaint.
